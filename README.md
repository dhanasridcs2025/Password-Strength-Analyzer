# Password Strength Analyzer 🔐

## Overview
This project is a Password Strength Analyzer developed using Python. It checks the strength of user-entered passwords based on length, complexity, and uniqueness.

The system also prevents password reuse by storing hashed passwords securely.

---

## Features
- Checks password length
- Detects uppercase, lowercase, numbers, and special characters
- Provides password strength rating
- Suggests improvements
- Uses SHA-256 hashing
- Prevents reuse of old passwords

---

## Technologies Used
- Python
- Regex
- SHA-256 Hashing
- File Handling

---

## How to Run

1. Open terminal in project folder
2. Run:

```bash
python password.py
```
